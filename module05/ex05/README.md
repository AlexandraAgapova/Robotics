# Glam Bot Drifter

## Концепция
Робот - икона стиля на колесах.

## Уникальный паттерн движения
Программа управления реализует циклический сценарий и состоит из этапов:
1.  **Форсаж:** Резкое ускорение по прямой.
2.  **Дрифт:** Резкий занос боком с сохранением скорости.
3.  **Восьмерка:** Классическая фигура высшего пилотажа.
4.  **Реверс:** Движение задним ходом.
5.  **Торнадо:** Финальное вращение на месте перед повторением всего цикла.

## Запуск
``ros2 launch unique_robot.launch.py``
